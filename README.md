# RedBlackTree
---
Instructions

---
Console Tester class

The user opens RedBlackTreeConsoleTester.java file
The insertion for Right-Right case is uncommented by default
To test the other Insertion cases the user can comment out the current case and the uncomment the case that they would like to test
User selects run to execute the file and see the result of the tree in the console

---
Graph Panel

The user opens GraphPanel.java file and selects run to execute the GUI program
The user in the insert box can enter the number that they would like to add to the tree
When the user clicks insert or prsses the neter button then it will be added to the tree
If the user wants to empty the tree they can click the clear button which will delete all the nodes from the tree
The user can select from the dropdown what shape the nodes are to be. The options include circle, square or rounded square

---
Tester class

The user runs the JUnit Test class. The expected result is that all tests pass
